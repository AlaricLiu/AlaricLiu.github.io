Java标记

自定义注解

@NameBinding
@target({ElementType.METHOD, ElementType.TYPE})
@Retention(RetentionPolicy.RUNTIME)
public @interface TokenAuthenticated{}

绑定一个filter到一个特定的JAX-RS方法
使用方法：
1，定义注解

@NameBinding
@target({ElementType.METHOD, ElementType.TYPE})
@Retention(RetentionPolicy.RUNTIME)
public @interface TokenAuthenticated{}

2，绑定注解到特定的filter

@Provider
@PreMatching
@TokenAuthenticated
public class BearerTokenFilter implements ContainerRequestFilter {
...
}

3，使用：

@Path("/customers")
public class CustomerResource {
@GET
@Path("{id}")
@TokenAuthenticated
public String getCustomer(@PathParam("id") String id) {...}
}

当调用了getCustomer时，将自动先调用filter方法。

@interface 与 interface

@interface 定义一个 interface
interface 表示该函数为接口函数。

Fabric标记

@Scope

@Scope(“COMPOSITE”) 标记成Compsite的只有一个实例
DOMAIN(集群单例), COMPOSITE(RUNTIME单例), STATELESS(默认, 一请求一个实例)
CONVERSATION(会话一个实例，但是Restful是无状态的)

@EagerInit

@EagerInit 由tomcat加载的时候来实例化类。只能跟在@Scope(“COMPOSITE”)之后

@Property

@Property 注释指定注入服务组件配置中的配置属性。在composite.xml中写具体的变量。

@Reference

@Reference 引用的类，在代码中与XML中都需要配置(可视化SCA上显示为紫色的)。

RESTFul标记

@Path

@Path("/hello")
public interface orgService()

URL路径，表示orgService处理 baseurl/hello的请求。

@Consumers

@Consumers("application/json")

接收请求数据类型

@Producers

@Producers("application/xml")

应答数据类型

@PathParam

@path("/hello/{id}")
public interface orgService(@PathParam("id")int id)

例如前端url /url/hello/1

@QueryParam

@Path("/hello")
@QueryParam("id") int id,

例如前端的url为 url/hello?id=1

public interface orgService(@QueryParam("id") List<string> listid,
@QueryParam("user") List<string> listUser)

对应url为：url/hello?id=1&id=2&user=a&user=b
